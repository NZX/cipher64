# Cipher64

Blowfish encryption and decryption of strings

## Installation

Add this line to your application's Gemfile:

    gem 'cipher64'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cipher64

## Usage

Blowfish encryption and decryption of strings
